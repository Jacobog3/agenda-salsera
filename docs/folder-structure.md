# Folder Structure

```text
src/
  app/
  components/
  i18n/
  lib/
  types/
supabase/
  migrations/
  seed/
docs/
```

## Notes

- `app/` contains localized routes and route handlers.
- `components/` contains reusable UI and page sections.
- `i18n/` contains routing and translation messages.
- `lib/` contains Supabase clients, queries, metadata helpers, validators, and utilities.
- `types/` contains shared domain types.
- `supabase/` contains SQL migrations and seed data.
